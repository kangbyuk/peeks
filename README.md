## 🕵️ 몰래보기 모드 4종 — PEEKS의 핵심

상황에 맞춰 **한 번에 켜고 끄거나 조합**할 수 있습니다. *(설정 → 몰래보기 탭)*

| 모드 | 이렇게 쓰면 좋아요 |
|------|---------------------|
| **투명도** | 위젯 전체를 **10%～100%**로 조절합니다. 뒤에 열어 둔 문서·브라우저가 비치면 “그냥 배경”처럼 보입니다. |
| **고스트 모드** | 평소엔 **살짝 흐릿하게** 두고, **마우스만 올리면 선명**해집니다. 지나가는 사람 눈에는 덜 띄고, 볼 때만 또렷하게. |
| **모노크롬** | 화면 전체를 **흑백 위장**합니다. 스포츠 앱 같지 않고, 그냥 **단조로운 정보 창**처럼 보일 때가 많습니다. |
| **The Coder Mode** | 로고·그라데이션 대신 **모노스페이스 + 다크 톤**. 스코어를 **터미널 로그**처럼, 또는 **`const match = { … }`** 코드 블록처럼 표시할 수 있어 **개발자 책상**에 자연스럽게 녹습니다. *(모노크롬과 함께 켜면 코더 스타일이 시각적으로 우선합니다.)* |

네 가지를 **겹쳐 쓰면** “흐릿한 반투명 터미널 창” 같은 조합도 가능합니다. 몰래보기는 단순 숨기기가 아니라 **주변 시선·화면 구성에 맞춘 연출**입니다. *(적당히 쓰시고, 업무는 챙기세요.)*

## ✨ 주요 기능

- **경기 & 스코어** — 오늘 내 팀 카드, **ALL GAMES**로 리그 전체 스코어보드  
- **순위** — NBA 동·서부, MLB, 축구 리그별 테이블  
- **즐겨찾기** — 팀 등록, 카드 **드래그로 순서 변경**, 로고로 빠른 추가  
- **창** — 크기·위치 기억, **Esc 두 번**으로 최소화(보스 키 느낌)

## 🚀 설치 & 실행

### 일반 사용자

1. GitHub **[Releases](https://github.com/kangbyuk/peeks/releases)** 에서 macOS용 **`.dmg`** 를 받습니다.  
2. 앱을 **Applications** 폴더로 옮긴 뒤 실행합니다.  
3. macOS에서 **보안 경고**가 나오면 시스템 설정에서 **열기**를 허용하면 됩니다.

> 아직 릴리스가 없다면 아래 **개발자** 방법으로 실행하거나, 개발자에게 빌드를 요청하세요.

### 개발자

```bash
git clone https://github.com/kangbyuk/peeks.git
cd peeks
npm install
npm start
```

로컬에서 설치 파일을 만들려면 `npm run build` (산출물은 `dist/`).

### 유지보수: GA4 Measurement Protocol

앱은 **GA4 Measurement Protocol**로 `app_start`, `add_team`, `change_mode` 이벤트를 보낼 수 있습니다. 측정 ID는 `G-M16NPNDWPG`이며, **API 시크릿**은 Git에 넣지 마세요.

- **개발:** 터미널에서 `export GA4_API_SECRET='(GA4 관리 화면에서 발급)'` 후 `npm start`
- **DMG에 포함해 배포:** `ga4-secret.local.json.example`을 복사해 `ga4-secret.local.json`으로 두고 `api_secret` 값을 채운 뒤, `package.json`의 `build.files`에 `"ga4-secret.local.json"`을 추가하고 `npm run build` (이 파일은 `.gitignore`됨)
