- **안정성·유지보수**: Electron 실행 경로 정리, macOS 캐시 정리용 `npm run clean`, 기타 스텔스/빌드 관련 정리.

## English

- **Multilingual UI**: **Korean / English** in Settings → General; system strings are translated while team names, leagues, and sport labels stay in English.
- **UI/UX**: Reorganized settings tabs (Stealth / Teams / General) and improved spacing for longer Korean text.
- **Reliability**: Clearer Electron launch script, optional `npm run clean` for macOS cache cleanup, and ongoing stealth/build housekeeping.
