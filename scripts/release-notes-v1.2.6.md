## PEEKS v1.2.6

### Highlights
- **KBO**: 홈 탭 나의 팀 경기에서 **예정 선발 투수**(네이버 단건 스케줄 API) 표시.
- **MLB**: 리그(AL/NL) **전체 순위** 보기(승률·이름 기준 정렬) 및 관련 UI.
- **축구**: **FA Cup** 리그 항목(순위표 없음 시 안내), 월드컵/설정·스타일 등 이전 작업분 반영.
- **UI**: 전체 경기(ALL GAMES) MLB 행에서 팀 **전체 이름**이 여백을 쓰도록 표시(말줄임은 창이 좁을 때만).

### Install
- `PEEKS-1.2.6-universal.dmg` — macOS (universal)
- `PEEKS-Setup-1.2.6.exe` — Windows installer
